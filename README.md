### プロダクト企画
- 企画からUIデザインまでを行い、数値的な成果もあげました。以下、マッチングアプリにおける自身が担当した企画/UIデザインとその成果を掲載します。
- 消費者視点を意識すること、既存の競合サービスから着想を得ること、数値での効果測定が可能であることを重要視しています。
- 初めてのマッチングから通話の実施率の向上（男性16.3%→25.0%, 女性17.4%→22.0%）
<details>
<summary>詳細</summary>

- 前提: 当アプリでは、マッチングした後に通話に進むことで課金に繋がるコンバージョンへと至る。そのためマッチングしてチャットをするだけではなく、通話を行うことが重要となっている。
- 課題: ユーザーごとの初めてのマッチングしたペアにおいて、通話に進む割合の向上（男性: 16.3%, 女性: 17.4%）
- 分析と仮説:
  - 実際の140組のユーザーのチャットを読み込んで分析した。
  - 男女の差が存在することを確認した（男性は35%の確率で通話の話題を出すが、女性は5%。男性の方が相手を誘いやすい傾向がある）。
  - 相手がすでに通話経験者であるかによっても大きく数値が異なることを確認。また、女性であっても自身が通話経験済みの場合は通話に誘うことを確認。
  - 会話内容を読み込むと、相手を通話に誘うことに緊張感を持っており、勇気を持って発信している様子が感じ取られた。
  - これらの内容から「相手が通話に関してどう感じているか分からず怖い」という課題がボトルネックであると特定した。また、「男性から女性を誘う & 通話経験済み女性から男性を誘う」という2つがメインのパスであると特定し、そのパスを促進する施策を打つこととした。
- 施策:
  - チャット欄を開いた際に、相手と通話をしたいと思っているかの回答を必須化
  - 回答を相手のチャット欄でサポートメッセージの形で表示する
  ![施策画像](/img/resume/talk_intention_sheet.png)
  *sheetのUIで回答を収集*
  ![施策画像](/img/resume/talk_intention.png)
  *あなただけに表示されています、として表示*
- 結果:
  - 男性にとっての初めてのペア
    |  | 施策前 | 施策後 |
    | --- | --- | --- |
    | 全体 | 16.3% | 25.0% |
    | 相手が通話経験無し | 12.4% | 22.6% |
    | 相手が通話経験あり | 23.9% | 28.0% |

  - 女性にとっての初めてのペア
    |  | 施策前 | 施策後 |
    | --- | --- | --- |
    | 全体 | 17.4% | 22.0% |
    | 相手が通話経験無し | 11.1% | 17.4% |
    | 相手が通話経験あり | 22.7% | 25.9% |
- 備考:
  - 意向確認という施策は、競合アプリが類似した機能を実装していたことからアイデアの着想を得ている。
</details>

- 身分証の認証率の向上
<details>
<summary>詳細</summary>

- 前提: 当アプリでは、マッチングをした相手とチャットを行うためには、インターネット異性紹介事業の規定により、身分証を提出して18歳以上であることを確認する必要がある。
- 課題: 身分証提出率が特に男性で低く、提出率は47.5%であった。
- 分析と仮説:
  - 消費者視点では身分証を出すこと自体はベネフィットを得られるわけではないので行いたくないアクションである上に、施策実施以前はモチベーションの設計を行っていなかった。（相手のチャット欄に入るとチャット画面ではなく身分証の提出を促す画面が表示される仕様だった）
  - 競合のマッチングアプリを研究したところ、「1通目のメッセージは送信可能、2通目以降は送信不可能、相手から届いたメッセージは身分証を認証しないと確認できない」という仕様であった。
  - 双方向のコミュニケーションが発生していないためインターネット異性紹介事業の規定を守りつつ、「実際に相手からメッセージが来たメッセージを見たい気持ち」や、「実際に自分がメッセージを送信する成功体験を味わいつつも、追加で送れないもどかしい気持ち」を抱かせることで、提出のフックとすることに成功していると判断した。
  - 自社アプリでもこの点についてはイノベーションの必要はなく、トレースして実装することが最も成功の確実性が高いと判断して企画した。
- 施策:
  - ペアとしての1通目は送信可能とする。
  - 相手から来たメッセージは確認できない。
  ![](/img/resume/identification_submission.png)
- 結果:
  | 性別 | 施策前 | 施策後 |
  | --- | --- | --- |
  | 男性 | 47.5% | 54.2% |
  | 女性 | 61.7% | 60.0% |

</details>
